# Security Features of Hedera Hashgraph

Hedera ensures **enterprise-grade security** using **Asynchronous Byzantine Fault Tolerance (aBFT)**.

## Key Security Features:

- **aBFT Consensus**: Proven to resist malicious attacks.
- **DDoS Protection**: Network remains stable under load.
- **Fair Ordering**: Transactions are processed fairly.

Hedera is **one of the most secure DLTs**, making it ideal for **financial applications and enterprises**.
